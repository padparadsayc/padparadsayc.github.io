doggo ​

doggo是一个用Golang编写的现代命令行DNS客户端

本教程演示了 doggo 的基本功能和用法:

  1. 查询指定类型的 DNS 记录：
    • 如果不指定记录类型（A, MX, NS ），则默认查询 A 类型 DNS 记录
    • 使用 -t <type> 指定查询的 DNS 记录，或者，你也可以省略 -t 参数，直接输入记录的类型。
  2. 使用 URL 类型的方案指定协议：
    • 默认情况下，使用 UDP 发送查询，如果要使用 https，可以使用 @https://，具体命令请看第四个示例
    • 已知协议有：udp, tcp, https, tls, sdns, quic
  3. 通过指定 --json 参数来输出 JSON 格式的 DNS 记录。
  4. 使用 --time 参数显示从服务器响应所花费的时间。
